How the World Wide Web works

The main type of document on the World Wide Web is HTML. It has all the instructions that ties everything together, whether that is images, JavaScript, or other files.

The main components of the World Wide Web are: HTTP, the main protocol of the web; Servers, the computers that host the files that comprise the web; the Internet, the actual network; and the browser, the software that runs on a computer and displays files from the web.

File Types

Several types of files can be included on the web, and HTML files, as mentioned before, are the primary type of file. But other files may include images, PDFs, Word Documents, sound files, videos, or any other type of binary.

HTML Tags

HTML tags consider of a name and content. A bold tag, for example, consists of "greater than B less than", the text that needs to be bolded, and a closing tag, "greater than, slash, b, less than".

Similar tags exist for italics (em), underlining (u) and there are some tags that have attributes. Attributes can be used in CSS, but they can also be used in tags like the anchor tag for creating links like this one: YouTube

Stupid Computers

It is said that computers are stupid not because computers can't do smart, sophisticated things, but because computers will do what is explicitly programmed into them. This makes it so that often programmers will need to debug their code, which is to say they need to go back and review code to look for errors.
